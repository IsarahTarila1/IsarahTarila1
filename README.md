## 🔨 What I'm Building

### [Offabuy](https://offabuy.com) — WhatsApp Commerce Infrastructure
> *"Most buyers drop off. OffaBuy closes them."*

A full-stack commerce platform that automates the entire path from social media click to confirmed payment — inside WhatsApp. Built for Nigerian vendors who are tired of closing sales manually.

**Key features shipped:**
- Automated WhatsApp checkout bot via Business API
- Per-channel attribution (Instagram, TikTok, WhatsApp Status, paid ads)
- Drop-off Intelligence — AI-powered diagnosis of cart abandonment
- Exit intent surveys feeding a real-time analytics engine
- Paystack payment integration (card, bank transfer, USSD)
- Vendor dashboard with funnel analytics and product management
- Role-based access and session management
